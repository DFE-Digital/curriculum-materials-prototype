Objective: to learn the types of volcano

    To know the characterstics of different types of volcanoes, and distinguish one type from another based on its eruption style.

    To locate volcanoes on a map using grid coordinates.

    " }) }}

    Lesson overview

    Core knowledge for all pupils

    • volcanoes occur at a destructive plate boundary
    • a lava flow is molten rock only
    • a pyroclastic flow is a mixture of molten rock, ash and gases

    Vocabulary for this learning objective

    • cone
    • vent
    • magma chamber
    • dome
    • shield
    • strato

    Misconceptions to address

    The following common misconceptions that pupils have should be considered when planning this lesson:

    {{ govukAccordion({ id: "accordion-default", items: [ { heading: { text: "All volcanoes erupt violently" }, content: { html: "

    This misconception arises from media coverage which is biased by location and death tolls.

    " } }, { heading: { text: "If a volcano doesn’t erupt for a hundred years, it’s extinct" }, content: { html: "

    In fact, the volcano is probably just dormant.

    " } }, { heading: { text: "Volcanoes only form on land, not underwater" }, content: { html: "

    Actually, there are numerous oceanic fault lines with volcanoes forming along these without breaking the surface of the water.

    " } }, { heading: { text: "Magma comes from deep within the Earth’s mantle" }, content: { html: "

    This comes from textbooks’ saying that the magma is ‘deep’ down - in reality, virtually all magma originates in the upper portion of the mantle and crust because of plate friction.

    " } } ] }) }}

    Pupils coming into the lesson need to know

    Vocabulary

    • lava: red-hot liquid rock
    • volcano: a mountain that erupts, or has erupted
    • volcanic eruption: an explosion or outpouring of lava, rock or as from a volcano
